Docker Volume is a mechanism to persist data generated by containers.
👉 Normally, container data is lost when the container is removed
👉 Volumes allow data to survive container deletion


__________Commands_________

1. Mapping volume
- docker run -v /host/path:/container/path nginx
- docker run -it -v myvolume:/data ubuntu bash

2. Create volume
- docker volume create myvolume

3. Check volumes
- docker volume ls

4. Best approach to mount the volume
- Build image → Create container + mount volume → Run
- We cannot attached the volume in running container
